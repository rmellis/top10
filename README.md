# 🏆 Top 10 Hub

Welcome to the **Top 10 Hub** repository. This project serves as a static hosting platform for a collection of highly curated, informative "Top 10" lists intended for general utility. 

Our inaugural guide focuses on the desktop publishing space, helping users transition away from deprecated legacy software.

## 📑 Current Lists

* **[Top 10 Microsoft Publisher Alternatives (2026)](./index.html)**
  A comprehensive, SEO-optimized guide ranking the best free, paid, and open-source desktop publishing software. Features a deep dive into options like OpenPublisher, Adobe InDesign, and Scribus.

## ✨ Technical Features

The lists in this repository are built with a focus on speed, aesthetics, and modern web standards.

* 🎨 **Windows 11 Fluent Design:** Soft acrylic blurs, responsive window panels, and modern hover effects.
* 🌓 **Native Dark Mode:** Instant light/dark theme toggling built directly into the UI.
* 🚀 **Zero Dependencies:** Pure, vanilla HTML, CSS, and JavaScript. No external frameworks required (excluding Font Awesome for iconography).
* 📱 **Fully Responsive:** Optimized layouts for both desktop and mobile viewing.
* 🛡️ **AdBlocker-Safe Promos:** Custom-built internal promotion system designed to seamlessly display web apps without triggering ad-blocking extensions.

## 🚀 How to Run

Because this project is built entirely with vanilla web technologies, there is no build process or server required.

1. Clone or download the repository.
2. Open `index.html` directly in any modern web browser.
